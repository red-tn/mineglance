# MineGlance

Mining profitability dashboard for cryptocurrency miners.

## Features

- Real-time mining stats from 2Miners pools
- Multi-coin support (RVN, ERGO, etc.)
- Net profit calculations with electricity costs
- Chrome extension for quick monitoring

## Pricing

- **Free** - 1 pool, 1 coin, manual refresh
- **Pro ($29)** - Unlimited pools/coins, auto-refresh, notifications
- **Pro + Mobile ($59)** - Everything in Pro + upcoming iOS/Android app

## Tech Stack

- Next.js 14
- Tailwind CSS
- Stripe (embedded checkout)
- Supabase
- Chrome Extension (Manifest V3)
